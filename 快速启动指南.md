# 🎮 网页端水果忍者 - 快速启动指南

## 方式 1: 运行主游戏 (推荐)

### 步骤 1: 编译 TypeScript 代码
```bash
npm run build
```

### 步骤 2: 启动 HTTP 服务器
```bash
npm run serve
```

### 步骤 3: 打开浏览器
访问: http://localhost:8080

**注意**: 
- 摄像头功能需要 HTTPS 或 localhost
- 首次访问会请求摄像头权限，请允许
- 如果摄像头不可用，游戏会自动切换到鼠标模式

---

## 方式 2: 运行端到端测试

### 步骤 1: 编译代码（如果还没编译）
```bash
npm run build
```

### 步骤 2: 启动服务器（如果还没启动）
```bash
npm run serve
```

### 步骤 3: 打开测试页面
访问: http://localhost:8080/e2e-test.html

点击"运行测试"按钮即可执行完整的测试套件。

---

## 一键启动（推荐新手）

如果你想一次性完成编译和启动服务器：

```bash
npm start
```

这个命令会：
1. 自动编译 TypeScript 代码
2. 启动 HTTP 服务器在 8080 端口

然后在浏览器中访问:
- **主游戏**: http://localhost:8080
- **E2E 测试**: http://localhost:8080/e2e-test.html

## 游戏控制

### 摄像头模式（默认）
- 将手放在摄像头前
- 移动手指来切割水果
- 避免切到炸弹

### 鼠标模式（降级模式）
- 按住鼠标左键并移动来切割水果
- 如果摄像头不可用会自动启用

### 触摸模式（移动设备）
- 用手指在屏幕上滑动来切割水果

## 常见问题

### Q: 摄像头无法访问？
A: 
1. 确保使用 HTTPS 或 localhost
2. 检查浏览器权限设置
3. 游戏会自动切换到鼠标模式

### Q: 页面显示空白？
A: 
1. 检查是否运行了 `npm run build`
2. 确保使用 HTTP 服务器（不要直接打开 HTML 文件）
3. 打开浏览器控制台查看错误信息

### Q: 手势识别不准确？
A: 
1. 确保光线充足
2. 手部完全在摄像头视野内
3. 背景尽量简单

### Q: 如何重新观看教程？
A: 在主菜单点击"游戏教程"按钮
