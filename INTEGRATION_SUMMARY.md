## 实现的系统连接

### 1. 手势识别 → 碰撞检测
**文件**: `src/game/GameLoop.ts`

- 手势追踪器提供手部轨迹数据
- 碰撞检测器接收轨迹并检测与游戏对象的碰撞
- 支持鼠标降级模式作为备选输入方式

### 2. 游戏事件 → 音频播放
**文件**: `src/main.ts`

连接的音效:
- 切割水果 → `slice` 音效
- 切割炸弹 → `explosion` 音效
- 错过水果 → `miss` 音效
- 游戏结束 → `gameOver` 音效
- 按钮悬停 → `hover` 音效
- 按钮点击 → `click` 音效
- 游戏开始 → `gameStart` 音效 + 背景音乐

### 3. 游戏状态 → UI 更新
**文件**: `src/game/GameLoop.ts`

实时更新:
- 分数变化 → HUD 分数显示
- 生命值变化 → HUD 生命值显示
- 最高分更新 → HUD 最高分显示
- 游戏结束 → 游戏结束界面

### 4. 教程系统 → 游戏流程
**文件**: `src/main.ts`, `src/game/GameLoop.ts`

集成功能:
- 首次访问自动启动教程
- 教程步骤完成反馈
- 教程模式下的特殊游戏逻辑
- 教程完成后恢复正常游戏
- 主菜单提供重新观看教程选项

### 5. 教程系统 → 对象生成器
**文件**: `src/game/ObjectSpawner.ts`, `src/game/GameLoop.ts`

- 教程模式可控制只生成水果或炸弹
- 手势识别步骤不生成任何对象
- 教程完成后恢复正常生成逻辑

### 6. 错误处理器 → 降级模式
**文件**: `src/core/ErrorHandler.ts`, `src/game/GameLoop.ts`

- 摄像头错误自动显示友好提示
- 提供鼠标模式作为降级方案
- 游戏循环自动切换输入源
- 鼠标轨迹与手势轨迹使用相同接口
